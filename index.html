Animation Demo

Diese Demo soll dir demonstrieren wie oft ein Browser ein Element rendern muss, wenn die Animation nicht optimal gebaut wurde.

FPS

  1. Öffne diese Seite in Chrome
  2. Öffne die Developer Tools (z.B. mit F12)
  3. Öffne das "Performance" Fenster
  4. Mach die CPU 6x langsamer und nimm die Webseite auf. Führe anschließend die Animation aus.
  5. Schaue dir das Ergebnis der FPS an und vergleiche die gute und die schlechte Animation.

Rendering, Painting | Performance

  1. Öffne diese Seite in Chrome
  2. Öffne die Developer Tools (z.B. mit F12)
  3. Öffne das "Performance" Fenster
  4. Mach die CPU 6x langsamer und nimm die Webseite auf. Führe anschließend die Animation aus.
  5. Schaue dir die Zusammenfassung der Seite an. Vergleiche Rendering und Patining der beiden Animationen.

Rendering

  1. Öffne diese Seite in Chrome
  2. Öffne die Developer Tools (z.B. mit F12)
  3. Klicke auf die drei Punkte oben rechts. Dort auf "More Tools" > "Rendering"
  4. In dem neu geöffneten Fenster klicke auf die Optionen: "Paint flashing" (ganz oben).
  5. Öffne auf beiden Varianten das Side-Menü und vergleiche wie oft neu gerendert werden muss.
