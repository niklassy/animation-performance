Animation Demo

Diese Demo soll dir demonstrieren wie oft ein Browser ein Element rendern muss, wenn die Animation nicht optimal gebaut wurde. Mach dazu folgendes:

  1. Öffne diese Seite in Chrome
  2. Öffne die Developer Tools (z.B. mit F12)
  3. Klicke auf die drei Punkte oben rechts. Dort auf "More Tools" > "Rendering"
  4. In dem neu geöffneten Fenster klicke auf die Optionen: "Paint flashing" (ganz oben).
  5. Vergleiche wie oft und wann das Side-Menü hervorgehoben wird bei beiden Varianten.
